Introduction

LinearOperators.jl is a package for matrix-like operators. Linear operators are defined by how they act on a vector, which is useful in a variety of situations where you don't want to materialize the matrix.

using LinearOperators

prod(v) = [3v[1] - v[2]; 2v[1] + 2v[2]]
A = LinearOperator(
  Float64, # element type
  2,       # number of rows
  2,       # number of columns
  false,   # symmetric?
  false,   # Hermitian?
  prod     # Function defining v → Av
)

@show A
A * ones(2)
A = Linear operator
  nrow: 2
  ncol: 2
  eltype: Float64
  symmetric: false
  hermitian: false
  nprod:   0
  ntprod:  0
  nctprod: 0


2-element Array{Float64,1}:
 2.0
 4.0

The operator can be symmetric and/or Hermitian, and products with its transpose and adjoint can be defined as well.

prod(v) = [3v[1] - v[2]; 2v[1] + 2v[2]]
tprod(v) = [3v[1] + 2v[2]; -v[1] + 2v[2]]

A = LinearOperator(Float64, 2, 2, false, false, prod, tprod)

transpose(A) * ones(2)
2-element Array{Float64,1}:
 5.0
 1.0

Notice that since A defined above is real, then the adjoint transpose is inferred to also be tprod:

A' * ones(2)
2-element Array{Float64,1}:
 5.0
 1.0

In the following example we define a complex operator using the Fast Fourier Transform and its inverse. Notice that the operator is orthogonal.

using FFTW, LinearAlgebra

A = LinearOperator(16, 16, false, false, fft, nothing, ifft)

v = rand(16) + im * rand(16)
norm(A * v - fft(v)), norm(A' * v - ifft(v)), norm(Matrix(A' * A) - I)
(0.0, 0.0, 1.9812853513207183e-16)

Preallocated Operators

It is often useful to reuse the memory used in a linear operator. If the operator is created from a matrix A, PreallocatedLinearOperator(A) automatically creates the underlying memory to store A * v, transpose(A) * v and A' * v.

m, n = 50, 30
A = rand(50, 30)
op1 = PreallocatedLinearOperator(A)
op2 = LinearOperator(A)
v = rand(n)

op1 * v
al = @allocated for i = 1:100
  op1 * v
end
println("Allocation of op1: $al")
op2 * v
al = @allocated for i = 1:100
  op2 * v
end
println("Allocation of op2: $al")

Allocation of op2: 49600

Inverse Operator

Operators may be defined to represent (approximate) inverses.

A = rand(5,5)
A = A' * A
op = opCholesky(A)  # Use, e.g., as a preconditioner
v = rand(5)

norm(A \ v - op * v) / norm(v)

LBFGS Operator

A useful operator is the Limited BFGS operator, which implements the BFGS update with limited memory used in nonlinear optimization. This update comes in both direct (BkB_kBk​) and inverse (Hk=Bk−1H_k = B_k^{-1}Hk​=Bk−1​) form.

Application: Heat Equation

Consider a square plate of size L×LL \times LL×L with no heat exchange on the boundaries. The Heat Equation and boundary conditions related to the temperature u(t,x,y)u(t,x,y)u(t,x,y) at time ttt, position (x,y)(x,y)(x,y) is given by

∂u∂t=α(∂2u∂x2+∂2u∂y2), \frac{\partial u}{\partial t} = \alpha \bigg(\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2}\bigg), ∂t∂u​=α(∂x2∂2u​+∂y2∂2u​),

where α>0\alpha > 0α>0, and

∂u∂x(t,0,y)=∂u∂x(t,L,y)=∂u∂y(t,x,0)=∂u∂y(t,x,L)=0. \frac{\partial u}{\partial x}(t,0,y) = \frac{\partial u}{\partial x}(t,L,y) = \frac{\partial u}{\partial y}(t,x,0) = \frac{\partial u}{\partial y}(t,x,L) = 0. ∂x∂u​(t,0,y)=∂x∂u​(t,L,y)=∂y∂u​(t,x,0)=∂y∂u​(t,x,L)=0.

Discretizing ttt into [0,t1,…,tN][0,t_1,\dots,t_N][0,t1​,…,tN​] with δ=tn+1−tn\delta = t_{n+1} - t_nδ=tn+1​−tn​, x∈[0,L]x \in [0,L]x∈[0,L] into [x1,x2,…,xm][x_1,x_2,\dots,x_m][x1​,x2​,…,xm​] with h=xi+1−xih = x_{i+1} - x_ih=xi+1​−xi​ and yyy accordingly, with h=yj+1−yjh = y_{j+1} - y_jh=yj+1​−yj​, we define ui,jnu^n_{i,j}ui,jn​ as the approximation to u(tn,xi,yj)u(t_n,x_i,y_j)u(tn​,xi​,yj​). The derivatives can be approximated by finite differences:

∂u∂t(tn,xi,yj)≈ui,jn+1−ui,jnδ(Forward scheme in time);∂u∂x(tn,xi,yj)≈ui+1,jn−ui−1,jn2h(Central 1st order scheme);∂2u∂x2(tn,xi,yj)≈ui+1,jn−2ui,jn+ui−1,jnh2(Central 2nd order scheme);∂u∂y(tn,xi,yj)≈ui,j+1n−ui,j−1n2h(Central 1st order scheme);∂2u∂y2(tn,xi,yj)≈ui,j+1n−2ui,jn+ui,j−1nh2(Central 2nd order scheme). \begin{aligned} \frac{\partial u}{\partial t}(t_n,x_i,y_j) & \approx \frac{u^{n+1}_{i,j} - u^n_{i,j}}{\delta} \qquad \text{(Forward scheme in time)}; \\ \frac{\partial u}{\partial x}(t_n,x_i,y_j) & \approx \frac{u^n_{i+1,j} - u^n_{i-1,j}}{2h} \qquad \text{(Central 1st order scheme)}; \\ \frac{\partial^2 u}{\partial x^2}(t_n,x_i,y_j) & \approx \frac{u^n_{i+1,j} - 2u^n_{i,j} + u^n_{i-1,j}}{h^2} \qquad \text{(Central 2nd order scheme)}; \\ \frac{\partial u}{\partial y}(t_n,x_i,y_j) & \approx \frac{u^n_{i,j+1} - u^n_{i,j-1}}{2h} \qquad \text{(Central 1st order scheme)}; \\ \frac{\partial^2 u}{\partial y^2}(t_n,x_i,y_j) & \approx \frac{u^n_{i,j+1} - 2u^n_{i,j} + u^n_{i,j-1}}{h^2} \qquad \text{(Central 2nd order scheme)}. \end{aligned} ∂t∂u​(tn​,xi​,yj​)∂x∂u​(tn​,xi​,yj​)∂x2∂2u​(tn​,xi​,yj​)∂y∂u​(tn​,xi​,yj​)∂y2∂2u​(tn​,xi​,yj​)​≈δui,jn+1​−ui,jn​​(Forward scheme in time);≈2hui+1,jn​−ui−1,jn​​(Central 1st order scheme);≈h2ui+1,jn​−2ui,jn​+ui−1,jn​​(Central 2nd order scheme);≈2hui,j+1n​−ui,j−1n​​(Central 1st order scheme);≈h2ui,j+1n​−2ui,jn​+ui,j−1n​​(Central 2nd order scheme).​

ui,jn+1=(1−4γ)ui,jn+γ(ui−1,jn+ui+1,jn+ui,j−1n+ui,j+1n), u^{n+1}_{i,j} = (1 - 4\gamma)u^n_{i,j} + \gamma\Big(u^n_{i-1,j} + u^n_{i+1,j} + u^n_{i,j-1} + u^n_{i,j+1}\Big), ui,jn+1​=(1−4γ)ui,jn​+γ(ui−1,jn​+ui+1,jn​+ui,j−1n​+ui,j+1n​),

Implementing

Instead of creating the full matrix AAA, we create AAA as an operator to save memory.

function HeatEquationOp(L, m, δ, α)
  h = L / (m - 1)
  γ = α * δ / h^2
  κ = 1 - 4γ

  Tprod(v) = [κ * v[1] + 2γ * v[2];
             [γ * v[i-1] + κ * v[i] + γ * v[i+1] for i = 2:m-1];
              κ * v[m] + 2γ * v[m-1]]

  T = LinearOperator(Float64, m, m, false, false, Tprod)
  D = opEye(m) * γ

  function prod(v)
    Hv = zeros(m^2)
    Hv[1:m] .= [T  2D] * v[1:2m]
    for i = 2:m-1
      Hv[(i-1)*m+1:i*m] .= [D T D] * v[(i-2)*m+1:(i+1)*m]
    end
    Hv[end-m+1:end] .= [2D T] * v[end-2m+1:end]
    return Hv
  end

  return LinearOperator(Float64, m^2, m^2, false, false, prod)
end

The next function creates a HeatEquationOp operator and the linearized vector U with starting values.

function HeatEquation(u0, L, m, δ, α)
  h = L / (m - 1)
  U = zeros(m^2)
  for i = 1:m
    x = (i - 1) * h
    for j = 1:m
      y = (j - 1) * h
      U[(i - 1)*m + j] = u0(x, y)
    end
  end
  A = HeatEquationOp(L, m, δ, α)

  return U, A
end
